## アプリの説明

### プロモーションテキスト（170文字以内）
```
「やらないこと」を管理して、より良い習慣を身につけましょう。シンプルな操作で悪習慣を記録・追跡し、自己改善をサポートします。
```

### 説明文（4000文字以内）
```
「Not Todo」は、あなたの「やらないこと」を管理するためのシンプルなアプリです。

日々の生活で避けたい習慣や行動をリスト化し、つい違反してしまった時には記録することで、自分自身の行動パターンを把握し、改善につなげることができます。

【主な機能】

■ やらないことリスト
避けたい習慣や行動を登録できます。カテゴリ別に整理でき、永続的な目標と期間限定の目標を設定できます。

■ 違反記録
つい「やらないこと」をやってしまった時、ボタンひとつで記録できます。メモを残すこともでき、振り返りに役立ちます。

■ 統計機能
違反の傾向をグラフで可視化。曜日別・時間帯別の分析で、自分の弱い時間帯を把握できます。

■ リマインダー
毎日または週1回、やらないことを振り返るリマインダーを設定できます。

【こんな方におすすめ】
・悪い習慣を断ち切りたい方
・自己改善に取り組みたい方
・シンプルな習慣管理アプリを探している方
・データに基づいて行動を振り返りたい方

【プライバシーに配慮】
すべてのデータはお使いのデバイス内にのみ保存されます。外部サーバーへの送信は一切行いません。

シンプルだけど効果的。「Not Todo」で、やらない習慣を身につけましょう。
```
